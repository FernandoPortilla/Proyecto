¿Qué es Gestión Escolar?

Es una aplicaión web creada para aquellos que necesiten llevar un control de su vida escolar, esta herramienta te permitira subir tus calificaciones a su base de datos para que después puedan ser consultadas, ademas de proporcionarte datos como el promedio general, individual de cada una de las materias.

Fácil de usar

Con esta herramienta solo basta con rellenar unos formularios simples y el resto es realizado automáticamente.

Has click en ingresar información para empezar!
